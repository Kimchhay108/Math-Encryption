Caesar Cipher

Source: 101Computing.net

"One of the earliest and most well-known forms of cryptography was developed by Emperor Julius Caesar. His solution was simple but effective: shift each letter in the alphabet by a fixed number of positions. This technique came to be known as the Caesar Cipher." - Source: caesarcipher.net

The Caesar Cipher encrypts a message by shifting each letter forward a fixed number of positions in the alphabet. Here's a brief example with a shift of k = 7:

  1. Convert Numbers for each letter: A=0, B=1, C=2, D=3, E=4, F=5, G=6, H=7, I=8, J=9, K=10, L=11, M=12, N=13, O=14, P=15, Q=16, R=17, S=18, T=19, U=20, V=21, W=22, X=23, Y=24, Z=25.
  2. Encrypt Formula: The Caesar Cipher encrypts a letter by shifting its number value by k and wrapping around using C = (P + k) mod 26.
  3. Example: If plaintext is HELLO and key = 7:
    H (7) + 7 = 14 ‚Üí O
    E (4) + 7 = 11 ‚Üí L
    L (11) + 7 = 18 ‚Üí S
    L (11) + 7 = 18 ‚Üí S
    O (14) + 7 = 21 ‚Üí V
  4. Result: The ciphertext becomes OLSSV.

Decryption is simply the reverse: P = (C - k) mod 26.

RSA Encryption Algorithm
  1. Step 1: Convert the message (M) into a number using ASCII or other encoding schemes.
  2. Step 2: Use the public key (e, n) to encrypt the message using the formula:
    C = Me mod n
RSA Encryption Example

Public Key: e = 3, n = 33
Message (M): 4

Encryption Formula:
C = Me mod n = 43 mod 33 = 64 mod 33 = 31

RSA Decryption Demo

RSA Decryption is the process of converting encrypted data (cipher text) back into its original readable form (plain text) using the private key. Once the recipient receives the cipher text, they apply their private key to decrypt it, ensuring that only the rightful owner‚Äîwho holds the private key‚Äîcan access the original message.
